# Pupervisor Web UI

Web UI для управления процессами супервизора (supervisor).

## Описание

Pupervisor - это веб-интерфейс для управления процессами, запущенными под контролем супервизора. Интерфейс позволяет просматривать статус процессов, запускать, останавливать и перезапускать их, а также просматривать логи.

## Функциональность

- Просмотр статуса процессов (запущен, остановлен, приостановлен)
- Управление процессами (запуск, остановка, перезапуск)
- Просмотр логов в реальном времени
- Автоматическое обновление информации каждые 30 секунд

## Требования

- Go 1.24 или выше
- gorilla/mux (автоматически устанавливается через go mod)

## Установка и запуск

1. Убедитесь, что Go установлен в системе
2. Склонируйте репозиторий
3. Установите зависимости:
   ```bash
   go mod tidy
   ```
4. Запустите сервер:
   ```bash
   go run cmd/server/main.go
   ```
5. Откройте браузер и перейдите по адресу: http://localhost:8080

## API endpoints

- `GET /api/processes` - получить список процессов
- `POST /api/processes/{name}/start` - запустить процесс
- `POST /api/processes/{name}/stop` - остановить процесс
- `POST /api/processes/{name}/restart` - перезапустить процесс
- `GET /api/logs` - получить логи

## Скриншоты интерфейса

Интерфейс включает:

1. Секцию статуса процессов с индикаторами состояния
2. Кнопки управления для каждого процесса
3. Область просмотра логов с цветовой кодировкой уровней важности
